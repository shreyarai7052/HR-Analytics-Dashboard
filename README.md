# 📊 HR Analytics Dashboard | Power BI

## 📌 Overview

The **HR Analytics Dashboard** is an interactive Power BI project designed to analyze employee data and provide insights into workforce trends, employee attrition, demographics, and organizational performance. The dashboard enables HR teams and business leaders to make data-driven decisions by identifying key patterns and improving employee retention strategies.


## 🎯 Project Objective

The objective of this project is to:

* Analyze employee demographics and workforce distribution.
* Monitor employee attrition and retention.
* Identify departments with high employee turnover.
* Evaluate workforce diversity and experience levels.
* Build an interactive dashboard that supports HR decision-making.


## 🛠️ Tools & Technologies

* **Power BI Desktop**
* **Power Query** (Data Cleaning & Transformation)
* **DAX (Data Analysis Expressions)**
* **CSV Dataset**

## 📂 Dataset

The dataset contains employee information, including:

* Employee ID
* Gender
* Age
* Department
* Job Role
* Education
* Marital Status
* Salary
* Years at Company
* Attrition Status
* Performance Rating


## 📊 Dashboard Features

The dashboard includes:

* Employee Overview
* Total Employees KPI
* Attrition Analysis
* Attrition Rate
* Department-wise Employee Distribution
* Gender Distribution
* Age Group Analysis
* Education Analysis
* Job Role Analysis
* Interactive Filters (Slicers)

## 📈 Key Insights

* Identified departments with the highest employee attrition.
* Compared employee distribution across departments.
* Analyzed workforce demographics by age and gender.
* Evaluated employee retention trends.
* Highlighted key HR metrics for better workforce planning.
